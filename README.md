# MusicNest

A personal music hub to import, organize, and play your music collection.

## Features

- **Import Music**: Support for MP3, MP4, WAV, FLAC, and more.
- **Organization**: Automatically organizes by artist and album.
- **Playlists**: Create and manage custom playlists.
- **Offline Mode**: Download imported music for offline playback.
- **Responsive Design**: Works seamlessly on desktop and mobile.
- **Modern UI**: Built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Shadcn UI
- **State Management**: Zustand
- **Database**: Dexie.js (IndexedDB)
- **Animations**: Framer Motion

## Getting Started

1. Clone the repository
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open http://localhost:8080 to view it in the browser.
